## Requirements:
* [NodeJs](https://nodejs.org/en/) (LTS / Current) 


## Installation:
* [Clone this repo](https://github.com/achwin/api-disease-spread.git)
* Run command berikut di cmd direktori repo tadi
```bash
npm install
```
* Setelah itu run command berikut
```bash
nodemon
```
